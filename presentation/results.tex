        \end{tabular}

    \begin{itemize}
      \item Best performance: RFR with Unrestricted (0.757)
      \item Syntactic features less informative than Lexical/Strings
      \item Feature combination improves SVR/RFR
      \item MLP suffers from overfitting
    \end{itemize}
